# Ms8_Espadon
This board has 8 outputs, 5 as PWM output and 3 as digital outputs.  
It read 8 or 16 RCUL (Xany) commands.  
It's use a little [Mini Nano V3](https://www.google.com/search?client=firefox-b-d&channel=entpr&q=arduino+mini+nano+v3) .
